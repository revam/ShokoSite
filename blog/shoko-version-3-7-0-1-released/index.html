Shoko Version 3.7.0.1 Released

Shoko >> Blog >> Shoko version 3 7 0 1 released

Posted On: December 29, 2016

Banner Anime: Sket Dance

Categories: General | Update
Tags: My Anime 3 | Shoko Desktop | Shoko Server | Web UI

We’ve released a maintenance update, version 3.7.01 for the following programs/plugins.

  • Shoko Server
  • Shoko Desktop
  • My Anime 3

This maintenance update should resolve all migration issues, JMM uninstall issues and also includes some additional bug fixes. For a full list of changes please check the changelog . If you are still encountering any migration issues on 3.7.0.1, please contact us on Discord.
